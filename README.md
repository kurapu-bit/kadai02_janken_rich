# ①課題名
ボイスじゃんけん（近畿編）

## ②課題内容（どんな作品か）
- 声で操作するじゃんけんゲーム。音声認識で対戦し、勝敗を記録できるアプリ
- 今回は、勝ったら青色、負けると赤色に近畿地方の地図が塗られていくように追加の機能を実装しました

## ③アプリのデプロイURL
後ほど貼り付け

## ④アプリのログイン用IDまたはPassword（ある場合）
- 無し

## ⑤工夫した点・こだわった点
- ベクター画像を使って「地図」を実装しました
  - 四角をコードで描け、JavaScriptから色を変えたりイベント追加をおこなえ自由度があるため活用
- 勝利 or 敗北した時だけ県ごとの豆知識を追加できるようにした
  - 県ごとに簡易的な情報データベースを構築
- ランダムで1県を選ぶ処理

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 難しかった点
  - 新たに機能実装をするだけで、条件分岐の調整をしないといけなくなった点
  - また、追加のコードを入れる場所によってはエラーが発生して、UIに反映されないことが多々起きた（この解消に時間を多く費やしてしまった）
